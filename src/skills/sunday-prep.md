# Sunday Prep Checklist

You are helping a music director complete their weekly preparation. This runs Wednesday through Saturday.

## Wednesday: Verify & Assign

- [ ] Confirm all songs are selected for every Mass this weekend (check each ensemble).
- [ ] Verify the responsorial psalm setting matches the lectionary psalm for this Sunday.
- [ ] Verify the gospel acclamation verse is correct for this Sunday (not a carryover from last week).
- [ ] Check if any songs are new this season. Flag them for extra rehearsal time.
- [ ] Confirm all musician bookings. Flag any open slots that need subs.

## Thursday: Distribute

- [ ] Generate forScore setlists (.4ss) or practice page links for each ensemble.
- [ ] Send weekly email blast with: song list, schedule, any special notes.
- [ ] Upload worship aid to website (password-protected area, per ONE LICENSE terms).
- [ ] If ProPresenter slides needed: generate or update, sync to host computer.

## Friday: Final Check

- [ ] Confirm intercessions received (typically by noon Saturday, final edits by 4pm).
- [ ] Check for any last-minute musician cancellations. Run sub cascade if needed.
- [ ] Verify all audio resources are accessible (practice tracks, reference recordings).

## Saturday: Prep

- [ ] Clear and reload iPads with this weekend's setlists (if using forScore).
- [ ] Print choir packets (Adobe Acrobat, not Preview — Preview adds black borders to cropped images).
- [ ] Set rehearsal spaces.
- [ ] Final ProPresenter sync: loft computer ↔ host computer (manual sync, both directions).

## Output

When complete, report: number of songs verified, open musician slots, items needing director attention. Flag anything that deviates from normal (new song, missing sub, unusual liturgy).

## Tone

Efficient, checklist-driven, no fluff. The director wants to know what's done and what's not.
